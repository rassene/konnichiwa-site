# Font Files

Self-hosted font files for the public site. These are not committed to Git (see `.gitignore`).

## Required files

Download from Google Fonts and place `.woff2` files here:

### Playfair Display
- `PlayfairDisplay-Regular.woff2`
- `PlayfairDisplay-Italic.woff2`
- `PlayfairDisplay-SemiBold.woff2`
- `PlayfairDisplay-Bold.woff2`

### DM Sans
- `DMSans-Regular.woff2`
- `DMSans-Medium.woff2`
- `DMSans-SemiBold.woff2`

### JetBrains Mono
- `JetBrainsMono-Regular.woff2`
- `JetBrainsMono-Medium.woff2`

## Download command

```powershell
# Use google-webfonts-helper or download manually from fonts.google.com
# https://fonts.google.com/specimen/Playfair+Display
# https://fonts.google.com/specimen/DM+Sans
# https://fonts.google.com/specimen/JetBrains+Mono
```
